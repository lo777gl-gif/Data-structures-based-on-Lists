# Assignment 2 - Algorithms and Data Structures 
This project contains an implementation of the logical data structure **Min-Heap (Minimum Heap), Mystack and MyQueue** in Java language. 

Min-Heap -> based on MyArrayList
MyQueue -> based on MyLinkedList
MyStack -> based on MyArrayList

Wasn't used standard Java collections ('java.util.*`).
